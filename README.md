# ✨ About the Project

**Ignite** is an interactive front-end animation where users can **swipe the matchstick** to ignite a realistic flame. Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**, this project demonstrates modern web animation techniques with smooth transitions, glowing fire effects, smoke particles, and interactive user input.
